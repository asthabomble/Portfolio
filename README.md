## 🚀 Featured Projects

### GreenSetu
A web-based environmental monitoring system that analyzes mangrove degradation in the Mumbai Metropolitan Region using GIS and satellite data.

**Tech Stack:**  
Google Earth Engine • Streamlit • Plotly • Pandas • Leaflet • Linear Regression

🔗 Live Demo: http://greensetu.streamlit.app

---

### MediPin
A location-based emergency assistance platform that helps users find nearby hospitals within a 5 km radius while providing real-time navigation and first-aid guidance.

**Tech Stack:**  
Spring Boot • Spring Security • OAuth2 • Hibernate • JPA • PostgreSQL • PostGIS • Leaflet.js

🔗 GitHub: https://github.com/asthabomble/MediPin
